走在上海街头

乙の方寸地
2021-05-05

因陪家人看病，再次来到上海。春日早晨，从宾馆走往医院————少有车辆的道路以及两旁成荫的树木————一公里左右的步行路程走出了散步的舒适，心情轻松了许多。多年前第一次来上海，也是这样冷暖交替的季节，那是毕业前的一次学术会议。

会议结束后的下午，换了一家快捷酒店安顿好行李，临近黄昏时出门，步行与一位事先约好的高中同学在附近碰头。已有八九年未见，彼此印象还停留在中学时代，虽常在熟识的人口中提起，见面后还是有些生疏。两人在附近一家饭店吃饭，之后在大学周边街道闲逛。比较有趣的，我走得太快被同学一把拉住，过往的自己过于不会体贴他人。期间在一家书店二楼点了饮品坐下休息聊天，聊的内容已没有太多印象，或者是关于高中时代的记忆、双方都认识的人，或是次日安排。

第二日看似匆匆的景点游，却是一位友人为差旅上海的我所做的精心准备，一早的会面与本地特色的早餐，徐家汇天主教堂偶遇的西式婚礼，田子坊的闲逛，傍晚街边的本帮菜馆与中途加入的友人，夜晚东方明珠附近地标建筑以及关于门票买单的小争执，外滩游览后公交车返程途中窗外的夜景，最终没有来得及看的电影在告别前留下的小遗憾。

时过境迁，多为故地重游时的感慨。但作为过客，再次来到上海时，竟也有这样的感觉。大概是几次与这座城市会面的时机，大都处于人生的路口，走过之后，一些不确定性消失不见，一些相聚别后难再重逢。
